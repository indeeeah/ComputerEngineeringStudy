## 실수 표현(부동 소수점, Floating-point)
* 과학적 표기의 지수(exponent)를 사용하여 소수점의 위치를 이동시킬 수 있는 표현 방법
* 표션의 범위가 확대
* 비트 수에 따른 분류가 가능하다
    + 단일 정밀도 부동 소수점 형식
    + S, 지수(E)필드, 기수(M)필드
    + ex) 0 0000101 1001 0000 0000 0000 0000
        - 부호 비트 : 0 -> +
        - 지수(E) = 0000101 = 5
        - 기수(M) = 1001 0000 0000 0000 0000 -> 0.1001*2`5
